# 🌍 War Sentiment Analysis using NLP

This project analyzes public sentiment on war-related topics (Iran–Israel conflict) using Natural Language Processing.

## 🚀 Features
- Sentiment Analysis using VADER (Positive, Negative, Neutral)
- Text Cleaning and Preprocessing
- Country-wise Sentiment Analysis
- Time-based Sentiment Trends
- Data Visualization using Matplotlib
- Interactive Dashboard using Streamlit

## 🛠️ Tech Stack
- Python
- Pandas
- VADER Sentiment
- Matplotlib
- Streamlit

## 📊 Sample Output
- Sentiment classification of tweets
- Graph showing sentiment distribution

## ▶️ How to Run
1. Install dependencies:
   pip install -r requirements.txt
2. Run main file:
    python main.py
   3. Run dashboard:
     streamlit run app.py

## 📌 Project Goal
To understand global public opinion on war scenarios using data analysis and NLP techniques.
